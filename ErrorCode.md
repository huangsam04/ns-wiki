## 开机黑屏报错
**A fatal error occurred while running Atmosphere.**
| Program ID        | 问题描述 / 解决方案                               |
|-------------------|-------------------------------------------------|
| 010000000000002B  | 系统可能因为极限超频损坏，重新创建虚拟系统即可。    |
